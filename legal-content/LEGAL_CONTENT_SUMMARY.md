# SpeakSync Legal Documents Content Summary

## Overview

This document provides comprehensive, user-friendly content for SpeakSync's legal documents that addresses all technical aspects of the application while remaining accessible to users.

## Key Features Covered

### Privacy Policy Highlights

**Data Collection:**
- Account information (email, name, profile data)
- Script content and metadata
- Voice and audio data (with ephemeral processing emphasis)
- Performance analytics (WPM, filler words, tone scores)
- Usage and technical data

**Data Processing:**
- Real-time speech-to-text for teleprompter highlighting
- Voice analysis for tone and emotion feedback
- AI-powered content suggestions
- Performance metrics calculation

**Third-Party Integrations:**
- **Deepgram** - Speech recognition
- **Hume AI** - Voice emotion analysis
- **Google Gemini** - Content enhancement
- **Stripe/RevenueCat** - Payment processing
- **Firebase** - Analytics and performance monitoring

**Key Privacy Commitments:**
- Ephemeral voice data processing (not stored beyond processing)
- Industry-standard encryption and security
- User control over data and privacy settings
- GDPR and CCPA compliance
- Clear data retention policies

### Terms of Use Highlights

**Service Description:**
- Professional teleprompter with AI features
- Real-time speech recognition and highlighting
- Cloud synchronization and analytics
- Voice analysis and performance feedback

**Acceptable Use:**
- Clear guidelines for permitted uses (speeches, presentations, education)
- Comprehensive prohibited uses (illegal content, IP infringement, abuse)
- Content guidelines emphasizing respect and legality

**Subscription Terms:**
- Free and premium tier structures
- Auto-renewal and cancellation policies
- Payment failure procedures
- Refund limitations

**Intellectual Property:**
- SpeakSync owns the application and technology
- Users retain ownership of their content
- Limited license for service processing

**Liability and Disclaimers:**
- "As is" service provision
- Speech recognition accuracy disclaimers
- Liability limitations appropriate for the service
- Force majeure provisions
